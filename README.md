# CRUD em Java

## 📖 Sobre o projeto

Este projeto foi desenvolvido durante uma atividade prática do curso Técnico em Desenvolvimento de Sistemas da ETEC de Guarulhos.

O desafio consistiu em implementar um sistema CRUD (Create, Read, Update e Delete) utilizando Java, aplicando os conceitos de Programação Orientada a Objetos, organização do código e lógica de programação. Todo o desenvolvimento foi realizado durante a aula, simulando um ambiente de resolução de problemas em tempo limitado.

## 🚀 Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)

## ✨ Funcionalidades

* Cadastro de registros
* Consulta de informações
* Atualização de dados
* Exclusão de registros
* Navegação pelo sistema através de menus

## 📚 Conceitos aplicados

* Classes e Objetos
* Encapsulamento
* Métodos
* Estruturas de decisão
* Estruturas de repetição
* Organização de código
* Operações CRUD

## 🎯 Objetivo

Praticar a implementação de um sistema CRUD utilizando Java, reforçando conceitos fundamentais da linguagem e desenvolvendo o raciocínio lógico na resolução de problemas.
